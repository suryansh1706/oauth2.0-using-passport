OAuth2.0 Authentication with Passport.js

A simple Node.js project demonstrating Google OAuth 2.0 authentication using Passport.js.
This project allows users to log in using their Google account and access a protected route.

Features : 
  1. Authenticate users via Google OAuth2.0
  2. Protected route that requires login
  3. Session management with express-session
  4. Uses environment variables for client ID and secret
  5. Beginner-friendly, easy to understand and extend

Tech Stack : 
  Node.js
  Express.js
  Passport.js (passport-google-oauth2)
  dotenv (for managing secrets)

Notes
  You can expand this project to use MongoDB or any other database for storing user sessions or profiles.
  This project is ideal for beginners learning OAuth2.0 authentication in Node.js.
